# Set Up and Launch a Personalization

Placeholder content for personalization setup documentation.

## Overview

Learn how to create and launch personalization campaigns that deliver tailored experiences to your visitors based on segments and behavior.

## Prerequisites

- Active Kameleoon account
- Website with Kameleoon script installed

## Steps

1. **Create a segment** - Define your target audience
2. **Design your personalization** - Create the personalized content
3. **Configure targeting rules** - Set when and where to show the personalization
4. **Launch** - Activate your personalization campaign
