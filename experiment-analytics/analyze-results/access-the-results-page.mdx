# Access the Results Page

Placeholder content for the results page documentation.

## Overview

Learn how to access and navigate the results page to analyze experiment performance.

## Accessing Results

1. Navigate to your experiment in the Kameleoon dashboard
2. Click on the **Results** tab
3. View metrics, conversions, and statistical significance

## Understanding the Results Page

- **Overview metrics** - Key performance indicators at a glance
- **Variation comparison** - Side-by-side performance of each variation
- **Statistical significance** - Confidence levels for the results
- **Goal performance** - How each goal is performing across variations
