---
title: "Custom Data"
description: "Overview of how custom data behaves in targeting, tracking, and reporting."
---

Custom data lets you attach business-specific values to a visitor for targeting and analytics use cases.

## Default logic when overwrite parameter is false or omitted

When overwrite is disabled or omitted, the SDK preserves the existing storage behavior defined for that custom data type and appends or merges values according to the implementation rules documented for the SDK you use.
