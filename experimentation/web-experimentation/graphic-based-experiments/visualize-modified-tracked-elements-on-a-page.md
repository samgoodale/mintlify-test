# Visualize modified/tracked elements on a page

You can track changes made to your site in your Kameleoon web experiments. This feature is particularly useful if you’re working in a large team with multiple modifications being made.

To use this feature, click the **cog** icon next to the element search bar in the Graphic editor.

![](../../../images/experimentation/web-experimentation/graphic-based-experimentation/visualize-modified-tracked-elements-on-a-page/elements-settings.png)

Check **Added elements** and **Modified elements** under **Highlight in preview** to highlight elements that have been added or modified.

You can now visualize which elements have been edited throughout your experiment's lifecycle. Elements that have been modified or added will now be labelled **Modified** or **Added** respectively in the left panel.

![](../../../images/experimentation/web-experimentation/graphic-based-experimentation/visualize-modified-tracked-elements-on-a-page/modified-added.png)
