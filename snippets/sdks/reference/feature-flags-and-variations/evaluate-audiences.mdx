* 📨 _Sends Tracking Data to Kameleoon_

This method evaluates visitors against all available Audiences Explorer segments and tracks those who match.

<props.ui.Code x={props.c.EvaluateAudiences.Name}/> should be called **after all relevant visitor data has been set or updated**, and **just before** getting a feature variation or checking a feature flag. This approach ensures that the visitor is evaluated against the most current data available, allowing for accurate audience assignment based on all criteria.

After calling this method, you can perform a detailed analysis of segment performance in Audiences Explorer.
