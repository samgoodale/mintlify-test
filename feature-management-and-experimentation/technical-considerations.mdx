## Streaming premium option

Streaming mode uses server-sent events (SSE) to push configuration changes to the SDK in real time, reducing the delay between a change in Kameleoon and its application in production.

If streaming is not enabled, SDKs continue to rely on their configured polling interval to fetch updated configurations.
